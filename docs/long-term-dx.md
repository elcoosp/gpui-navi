### Task 1.1: Fix `$` Filename Handling in Generator

**Files:**
- Modify: `navi-codegen/src/generator.rs`

- [ ] **Step 1: Update generator to emit `#[path]` attributes for route modules**

Currently `route_tree.gen.rs` uses `crate::routes::users::param_id::...` but the file is named `$id.rs`. Rust's module system cannot resolve this unless the file is renamed. We must emit a `#[path = "..."]` attribute to map the module ident to the actual filename.

```rust
// In generator.rs, build module declarations with path attributes
let mut module_decls = String::new();
for route in &routes {
    let mod_path = route.module_path.replace("::", "_");
    let file_path = route.relative_path.to_str().unwrap();
    module_decls.push_str(&format!(
        "#[path = \"{}\"] pub mod {};\n",
        file_path, mod_path
    ));
}
```

- [ ] **Step 2: Update the generated file template to include the module declarations**

```rust
let output = format!(
    r#"// AUTO-GENERATED by navi-codegen - DO NOT EDIT

pub mod _routes {{
{module_decls}
}}

pub fn build_route_tree() -> navi_router::RouteTree {{
    let mut tree = navi_router::RouteTree::new();
{route_nodes}
    tree
}}

pub fn register_routes(cx: &mut gpui::App) {{
{register_calls}
}}
"#,
    module_decls = module_decls,
    route_nodes = route_nodes,
    register_calls = register_calls,
);
```

- [ ] **Step 3: Update `example-app/src/main.rs` to use the new module structure**

```rust

```

            .map(|s| {
                let current = &s.current_location().pathname;

                    current == &self.to
                } else {

                }

            .children(self.children);

            if let Some(f) = self.active_style {
                element = f(element);
            } else {
                element = element.font_weight(FontWeight::BOLD);
            }
        } else if let Some(f) = self.inactive_style {
            element = f(element);
        }
        element
    }
}

```rust
use gpui::*;
use crate::RouterState;

pub struct SuspenseBoundary {
    fallback: Box<dyn Fn() -> AnyElement>,
}

impl SuspenseBoundary {
    pub fn new(fallback: impl Fn() -> AnyElement + 'static) -> Self {
        Self { fallback: Box::new(fallback) }
    }
}

impl RenderOnce for SuspenseBoundary {
    fn render(self, _window: &mut Window, cx: &mut App) -> impl IntoElement {
        let is_loading = RouterState::try_global(cx)
            .map(|s| s.has_pending_loader())
            .unwrap_or(false);
        if is_loading {
            (self.fallback)()
        } else {
            Outlet::new().into_any_element()
        }
    }
}
```
